# Global GDP Explorer

A professional, lightweight Python application for exploring Global GDP data using the World Bank API.

## 🚀 Features
- **Single Country Analysis**: Detailed GDP view for a specific country and year.
- **Comparison Tool**: Side-by-side economic trend analysis between two nations.
- **Growth Tracking**: Visualization of annual GDP growth percentages.
- **High Performance**: Built-in data caching for near-instantaneous chart switching.
- **Multi-Tasking**: View multiple charts and reports simultaneously in independent windows.
- **Sleek UI**: Modern dark theme with high-contrast visuals.

## 🛠️ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/gdp-explorer.git
   cd gdp-explorer
   ```

2. **Install the required libraries:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application:**
   ```bash
   python gdp_analyzer.py
   ```

## 📊 Data Source
All data is fetched in real-time from the [World Bank Open Data API](https://data.worldbank.org/).
